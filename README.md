# NewsFlash — Flutter News App

A beautiful, offline-capable Flutter news app powered by NewsAPI.

## ✨ Features
- Headlines Screen with featured card
- Detail Screen with hero image
- Offline Mode using Hive
- Category Filters
- Search
- Bookmarks
- Pull to Refresh
- Infinite Scroll
- Shimmer Loading
- Animations

## 🛠 Tech Stack
- provider
- http
- hive + hive_flutter
- cached_network_image
- connectivity_plus
- flutter_animate
- shimmer
- share_plus
- url_launcher
- google_fonts
- intl
